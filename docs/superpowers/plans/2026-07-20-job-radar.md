# 招考雷达 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建一个可部署到 GitHub Pages、每天自动聚合公务员、事业单位与央国企招聘公告，并支持搜索、筛选、收藏和直达原公告的网站。

**Architecture:** Python 标准库采集器从官方栏目读取公开链接，并支持按需接入搜索 RSS，经过官方域名校验、分类、日期解析和去重后写入 `data/jobs.json`。无依赖的静态前端加载该 JSON，在浏览器内完成快速筛选、排序、收藏和三种显示模式；GitHub Actions 每天运行采集器并提交数据，Pages 负责托管。

**Tech Stack:** Python 3.11+ 标准库、HTML/CSS/ES modules、Node.js 20+ 内置测试器、GitHub Actions、GitHub Pages

## Global Constraints

- 采集公开网页，不绕过登录、验证码或访问控制。
- 结果链接必须通过 `crawler/sources.json` 中的官方域名白名单。
- 单个来源失败时继续处理其他来源，并保留旧数据。
- 前端不使用第三方运行时依赖，移动端点击区域不小于 44px。
- 数据更新时间显示为 Asia/Shanghai 时区，公告日期保留 ISO `YYYY-MM-DD`。
- 网站只做信息聚合；报名资格、截止时间和最终安排均以原公告为准。

```python
def is_allowed_url(url: str, allowed_domains: list[str]) -> bool:
    parsed = urlparse(url)
    if parsed.scheme not in {"http", "https"} or not parsed.hostname:
        return False
    host = parsed.hostname.lower().rstrip(".")
    return any(host == domain.lower() or host.endswith("." + domain.lower()) for domain in allowed_domains)

def merge_with_previous(new_jobs: list[dict], previous: dict, failed_sources: set[str], now: datetime) -> list[dict]:
    if failed_sources and not new_jobs:
        return dedupe_jobs(previous.get("jobs", []))
    retained = [job for job in previous.get("jobs", []) if job.get("collector") in failed_sources]
    return dedupe_jobs([*new_jobs, *retained])
```

Implementation requirements: `urllib.request` with a 20-second timeout and user-agent; `xml.etree.ElementTree` for RSS; `html.parser.HTMLParser` for links; SHA-1 stable IDs; normalized title dedupe; ISO dates; temporary-file plus `Path.replace()` atomic output; non-zero exit only when every source fails and no old data exists.

export function formatRelativeDate(value, now = new Date()) {
  const days = ageInDays(value, now);
  if (!Number.isFinite(days)) return '日期待核';
  if (days <= 0) return '今天';
  if (days === 1) return '昨天';
  if (days < 7) return `${days}天前`;
  return value.slice(5).replace('-', '.');
}
